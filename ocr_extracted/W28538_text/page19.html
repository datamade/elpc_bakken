depth.
4. Run and cement production casing with volumes required to cover the Dakota formation (see table above and enclosed formation top estimates).
5. Test casing to 1500 psig. Drill cement and float equipment.
6. Drill horizontal section to proposed TD.

8. Install fraC string 9. Rig down and move out drilling rig.
